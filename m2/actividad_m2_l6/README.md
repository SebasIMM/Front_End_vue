# ¿Qué es jQuery y cuáles son sus principales ventajas?

jQuery es una biblioteca de JavaScript que simplifica el uso del lenguaje.
Permite escribir menos código para hacer cosas como animaciones, manejar eventos o modificar elementos HTML.

Sus ventajas son:

- Código más corto y fácil de entender.
- Compatibilidad con muchos navegadores.
- Amplia cantidad de funciones listas para usar.

# ¿En qué casos conviene usar jQuery frente a JavaScript puro?

Conviene usar jQuery cuando se quiere hacer algo rápido, con poco código.
También es útil si el proyecto ya usa jQuery.

# ¿Qué relación existe entre jQuery y Bootstrap?

Las versiones antiguas de Bootstrap (hasta la 4) usan jQuery para manejar menús, ventanas modales y otros efectos interactivos.
En las versiones nuevas (Bootstrap 5), ya no es necesario jQuery porque todo se hace con JavaScript puro.

# ¿Qué es un plugin y en qué situaciones es recomendable utilizarlo?

Un plugin es una extensión o complemento que agrega funciones extra a un programa o biblioteca (como jQuery).
Se recomienda usarlo cuando necesitas una función específica —por ejemplo, un carrusel, un calendario o una galería— y no quieres programarla desde cero.
